Obtenha Sua API Key do Google Gemini para utilizar o assistente

Para que o assistente funcione, você precisa de uma chave de API válida do Google Gemini. Siga estas etapas simples para gerá-la:

  1. Acesse o Google AI Studio

    Vá para o site: Google AI Studio e certifique-se de estar logado com sua conta Google.

  2. Crie sua API Key

    Clique na opção "Criar chave de API".

  3. Copie a API Key Gerada

    Após a geração, copie a chave de API que será exibida e a guarde em um local seguro (ela é pessoal).

  4. Insira a Chave no Campo Designado

    Volte para cá e cole a sua chave de API copiada no campo que a solicita.

  5. Comece a Usar o Assistente!

    Selecione um jogo e faça suas perguntas para o assistente!
